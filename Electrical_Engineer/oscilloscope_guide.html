Probe Setup Procedure — Do This Every Session
  1. Attach the ground clip first — clip to the nearest ground point on the board. A long ground lead acts as an antenna and adds noise.
  2. Check probe compensation — touch the probe tip to the CAL terminal on the scope. The square wave should have flat tops and bottoms. Adjust the trim capacitor on the probe body if it looks rounded (over-compensated) or has sharp peaks (under-compensated).
  3. Set probe attenuation in the scope menu — match the setting (1x or 10x) to the physical switch on the probe body. Mismatch causes incorrect voltage readings.
  4. Select AC or DC coupling — use DC coupling to see the full signal including DC offset. Switch to AC coupling when measuring small ripple on top of a large DC rail.
  5. Set Volts/Div — start large, then decrease until the waveform fills 4–6 divisions vertically without clipping.
  6. Set Time/Div — start with 1–2 complete cycles visible. For a 1 MHz clock: 1 µs/div shows 10 cycles.
  7. Adjust trigger — set to rising edge, level at 50% of the signal swing for a stable, locked display.
⚠ Never connect the scope ground clip to a non-ground test point on a live board. The ground clip is directly connected to the mains earth through the scope — connecting it to a signal net will short that net to earth and damage the board or scope.

How to Read This Display

Grid: 10 horizontal divisions (time) × 8 vertical divisions (voltage). Each division = one Time/Div or Volts/Div step.

Period (T): Measure time for one complete cycle — from rising edge to next rising edge using the horizontal cursors.

Frequency: f = 1/T. A waveform that takes 2 divisions at 1 µs/div has T = 2 µs → f = 500 kHz.

Amplitude: Count vertical divisions peak-to-peak, multiply by Volts/Div. 4 divs at 1 V/div = 4 V p-p.

Duty cycle (PWM): (High time / Period) × 100%. Use horizontal cursors to measure each state.

Power-On Bring-Up Sequence — First Test on a New Board
  1. Visual inspection first — check for obvious solder bridges, missing components, wrong polarity electrolytic caps. Do not power on if damage is visible.
  2. Check input resistance to GND (power off) — measure resistance from the main power input to GND. Less than 10Ω suggests a short. Investigate before applying power.
  3. Apply power through a current-limited supply — set current limit to 10–20% above expected max. If the supply folds back immediately, there is a short.
  4. Measure each power rail — probe every test point in sequence: input rail, regulator output, downstream rails (3.3V, 1.8V, 1.2V). Compare to schematic spec.
  5. Check GND reference — probe TP_GND to confirm 0.000V. A floating or elevated GND indicates a broken return path.
  6. Move to oscilloscope — once all DC rails are confirmed, switch to scope for clock signal verification and dynamic load testing.
Morning Equipment Setup Checklist
  • Oscilloscope powered on and warmed up (≥5 min before calibration-sensitive measurements)
  • Probe compensation verified on CAL terminal — square wave top and bottom are flat
  • Probe attenuation setting in scope menu matches physical probe switch (1x or 10x)
  • Multimeter battery level checked — low battery causes incorrect readings on resistance and diode mode
  • Multimeter probes in correct input jacks (V/Ω jack for voltage/resistance, A jack for current)
  • Schematic and BOM revision confirmed to match board revision number on silkscreen
  • ESD wrist strap connected and continuity verified
  • Bench power supply current limit set for the board under test
Board Test Completion Checklist — Before Signing Off
  • All power rails measured and within ±5% of nominal — values recorded in test log
  • Clock signals verified on scope — correct frequency, amplitude, and duty cycle
  • Communication bus signals checked (SPI/I²C/UART) — no missing edges or ringing
  • Boundary-scan run completed (if JTAG header present) — fault report exported
  • Any failures documented with: test point, instrument, measured value, expected value, screenshot
  • NCR written for any out-of-spec or failed condition — serial number recorded
  • Board label updated with test date, result (PASS/FAIL), and technician ID
  • Test log entry complete — board added to daily yield count
